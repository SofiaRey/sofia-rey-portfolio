## Overview

Single-page portfolio website for Sof√≠a Rey (multimedia designer) with an Apple-style scroll-driven video frame animation as the full-page background. Built with Bun + React + Tailwind CSS.

## Architecture

### Three-layer visual stack

1. **Canvas (z-0)** ‚Äî Fixed full-viewport `<canvas>` rendering pre-extracted JPEG frames
2. **Content (z-10)** ‚Äî Page sections with transparent backgrounds, floating over the canvas
3. **Noise overlay (z-9999)** ‚Äî Animated film-grain texture, `pointer-events: none`

### Frame Sequence System

- **Source:** `src/assets/video.mp4` (2560x1440, 27s, 30fps)
- **Extraction:** `bun run extract-frames` ‚Üí 280 JPEG frames at two resolutions
  - Desktop: 1920x1080 ‚Üí `public/frames/desktop/0001.jpg`‚Äì`0280.jpg`
  - Mobile: 960x540 ‚Üí `public/frames/mobile/0001.jpg`‚Äì`0280.jpg`
- **Breakpoint:** ‚â§768px uses mobile set

### Playback Modes

1. **Intro Playback (frames 0‚Äì51):** Auto-plays over 5 seconds on page load. Scroll is ignored during this phase. Doubles as loading buffer for scroll frames.
2. **Scroll-Driven Mode (frames 52‚Äì279):** After intro completes, frame index = `52 + (scrollY / maxScroll) * 228`. If user scrolled during intro, canvas jumps to correct frame on transition.

### Frame Loading Strategy

- **Phase 1:** Preload all 52 intro frames eagerly (parallel Image() requests). Intro starts only when all are loaded.
- **Phase 2:** Background-load 228 scroll frames during intro playback. If a scroll frame isn't loaded yet, show nearest available frame.

## Responsive Strategy

- **Desktop (>768px):** Full layout, 1920x1080 frames
- **Mobile (‚â§768px):** Single-column stack, 960x540 frames, hamburger nav ‚Üí full-screen overlay

## Typography & Color

- **Serif (headings):** Libre Caslon Text (italic) ‚Äî Google Fonts
- **Sans-serif (body):** Kedebideri ‚Äî Google Fonts
- **Black:** `#000000`
- **Red accent:** `#C12D2D`
- **Background:** `#FFFFFF`

## Animations

- **Role rotation:** Vertical slide, 3s per word, infinite loop
- **Noise overlay:** Animated canvas-generated grain, ~4% opacity, 100ms refresh
- **Process items:** Orbit in a circle (20s CSS rotation), counter-rotate to stay upright, scale to 1.25 on hover, circle pauses on hover
- **Content reveals:** Fade-up (opacity 0‚Üí1, translateY 20px‚Üí0) via Intersection Observer, triggered once at 20% viewport entry
- **Photo carousel (mobile):** CSS marquee animation, continuous loop, ~35s cycle

## Build & Deploy

- **Dev:** `bun --hot src/index.ts` (port 3002, HMR)
- **Build:** `bun run build` (static output to `dist/`)
- **Frames:** `bun run extract-frames` (requires ffmpeg, output gitignored)
- **Deploy:** Static hosting (Vercel/Cloudflare Pages). Build command: `bun run extract-frames && bun run build`
